# RAG Document QA

## Project Overview
This project is a production-ready foundation for a PDF-based RAG chatbot built with FastAPI. The current Phase 1 setup provides a clean modular structure for future API development, document ingestion, and retrieval workflows.

## Folder Structure
```text
rag-document-qa/
├── app/
│   ├── main.py
│   ├── rag.py
│   ├── config.py
│   └── utils.py
├── data/
│   └── sample.pdf
├── vectorstore/
├── requirements.txt
└── README.md
```

## Planned Features
- FastAPI-based API endpoints for chat interactions
- PDF document ingestion and preprocessing
- Vector store integration for retrieval-augmented generation
- Configuration management via environment variables
- Modular and extensible backend architecture
